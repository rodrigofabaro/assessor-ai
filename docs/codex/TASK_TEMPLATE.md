# Codex Task Template

## Governing rules
- Read and follow: [NON_NEGOTIABLES](./NON_NEGOTIABLES.md)

## Context
Provide relevant background, existing behavior, and constraints.

## Goal
State the outcome in one or two sentences.

## Required behaviour
List required behaviors and any invariants that must not change.

## Acceptance tests
List the exact commands or user flows that must pass.

## Files to touch
List expected files (add/remove/update).

## Definition of done
Checklist of what “done” means (functional + UX + docs).
