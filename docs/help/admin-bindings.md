# `/admin/bindings` Help

## Purpose

Bind assignments to locked brief/spec references used by grading.

## Main actions

- review binding status
- lock assignment-to-brief mappings
- confirm reference coverage before grading

## How to use

1. Confirm brief and spec are locked.
2. Apply/verify assignment binding.
3. Ensure submissions point to intended binding before grading.

## Important rule

Binding lock defines criteria universe for grading and audit reproducibility.
