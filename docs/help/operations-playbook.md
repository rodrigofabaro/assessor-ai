# Operations Playbook (`/help/operations-playbook`)

Last updated: 2026-02-20

This guide is the end-to-end operating sequence:

1. App settings and governance defaults
2. Spec/brief/rubric extraction + lock
3. Assignment binding
4. Upload and triage
5. Preview and commit grading
6. Assessor criterion overrides (when needed)
7. QA review and audit close-out
