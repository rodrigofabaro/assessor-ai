# Submissions List (`/submissions`)

Last updated: 2026-02-19

Use this page as the operations queue.

## Primary Actions

- filter by lane/status/search
- resolve student links
- run QA preview/commit
- retry failed rows after fixes
- open row detail for deep review

## Lane Intent

- `Auto-ready`: automation-safe
- `Needs Human`: manual review required
- `Blocked`: do not grade yet
- `Completed`: finished runs

## Rule

Treat this page as queue control, not grading detail. Use `/submissions/[submissionId]` for full evidence review.
